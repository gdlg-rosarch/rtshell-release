rtpwd

.. include:: ../common/docinfo_block.txt

Synopsis
========

rtpwd

Description
===========

Displays the current working directory in the RTC Tree.

.. include:: ../common/common_body.txt

Examples
========

::

  $ rtpwd

Print the current working directory.
